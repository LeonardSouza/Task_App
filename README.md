# Task App

Aplicativo web de tarefas (to‑do) feito em **React + TypeScript**, focado em uso individual, com dados salvos localmente no navegador.

## Como usar

- Instale as dependências:
  - `npm install`
- Rode em modo desenvolvimento:
  - `npm run dev`
- Acesse no navegador:
  - `http://localhost:5173`

Na interface, preencha o **título** da tarefa (obrigatório) e, se quiser, adicione descrição, data de vencimento e prioridade. Depois clique em **“Criar tarefa”** para ver a tarefa aparecer na lista. As tarefas ficam salvas mesmo ao recarregar a página.
